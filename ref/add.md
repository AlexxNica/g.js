Add two numbers.

    grob.add(5, 3);

The function takes any number of arguments:

    grob.add(1, 2, 3, 4);

No arguments returns the "addition identity" -- the value which has no effect in addition:

    grob.add();


## Parameters
- `a`: The first number.
- `b`: The second number.
- `...`
