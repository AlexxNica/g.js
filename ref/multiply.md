Multiply numbers.

    grob.multiply(5, 3);

The function takes any number of arguments:

    grob.multiply(1, 2, 3, 4);

No arguments returns the "multiply identity" -- the value which has no effect in multiplication:

    grob.multiply();

## Parameters
- `a`: The first number.
- `b`: The second number.
- `...`
